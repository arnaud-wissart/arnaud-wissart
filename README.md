# Arnaud Wissart — Développeur .NET Full-Stack

Je conçois des applications web robustes avec **.NET 8/9/10**, de l’API jusqu’au front en **React/TypeScript**, avec un focus constant sur la **qualité logicielle** (tests automatisés, CI, conventions) et l’industrialisation via **Docker** et pratiques **DevOps**.
